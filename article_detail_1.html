java 문자열에서 부분 추출하기

JAVA
2020-12-17 20:55:32
이회구
# 자바 문자열 분리해서 추출하기 ```java // charAt(문장 중에 인덱스 위치에 해당문자 추출) //인덱스 범위 0~6 String aaa = \"가나다라마바사\"; // 인덱스3을 출력하면 \"라\"가 출력됩니다. char data = aaa.charAt(3); System.out.println(data); //출력값 : 라 //subString (원하는 범위만큼 문자열 추출) // 인덱스 범위 0~6 String aaa = "가나다라마바사"; //2~5에 해당되는 문자열 부분 추출 String data = aaa.substring(2,5); System.out.println(data); //출력값: 다라마 //split(주어진 문자로 분리하여 배열에 저장) // 띄어쓰기를 포함한 문장 String aaa = "가나 다라마 바 사"; //split으로 띄어쓰기를 구분하여 배열에 저장 String[] data = aaa.split(" "); //구분된 배열을 배열의 길이만큼 출력 for (int i = 0; i < data.length; i++) { System.out.println(data[i]); } //출력값: 가나 다라 바 사 //split 2탄 //구분은 &와 ,(쉼표)와 -로 구분 String aaa = "안녕&하세요,반갑-습니다"; // &와 ,(쉼표)와 -로 구분해보면 String[] data = aaa.split("&|,|-"); for (int i = 0; i < data.length; i++) { System.out.println(data[i]); } //출력값: 안녕 하세요 반갑 습니다 ```
